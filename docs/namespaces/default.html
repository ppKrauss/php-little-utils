assoc_join()

assoc_join(  $a,   $sep = '; ',   $pairSep = '=') : \mix

Joins a set of key-value pairs by $pairSep, and all pairs of the set by $sep.

NOTE: assoc_join($a,'&') is similar to http_build_query($a).

Parameters

$a

mix NULL or array (handdled as reference) to be joined.
$sep

string (default '; ') separator in the final string.
$pairSep

string (default '=') pair separator (joins key-val pair).

Returns

\mix —

NULL if $a is NULL, string if $a is array.

xml2json()

xml2json(  $input,   $out_php = true,   $out_jsonml = true,   $out_array = true,   $flenLimit = 1000) : \JSON

Converts XML file (or XML string or DOMDocument) to JSON or to PHP array.

Use direct SimpleXML-JSON convertion or jsonML (see https://github.com/mckamey/jsonml).

Parameters

$input

string of filename or string of full XML or DOMDocument instance.
$out_php

boolean true if whant PHP output, else will be JSON.
$out_jsonml

boolean true if whant jsonML as outputformat, else will be SimpleXML-JSON.
$out_array

boolean true if whant pure PHP array as output, else will be array with objects.
$flenLimit

integer 0 or size limit for filename, only for safe.

Returns

\JSON —

string or array of XML mapped data.

NOTE: for other xml-json map strategies, see https://github.com/bramstein/xsltjson FUTURE function dom2json() can be used with XPath to select specific node or array of nodes.
