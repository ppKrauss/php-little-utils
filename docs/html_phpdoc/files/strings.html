str_getRoot()

str_getRoot(  $a,   $b = '',   $minLen = 1,   $retFamily = false) : string

Get the commom start-string of two or more strings.

The algorithm splits the string by its differences.

Parameters

$a

mix, string for final operation, or array iteration.
$b

string with probable commom root, or integer for array index.
$minLen

integer minimal length.
$retFamily

boolean true for return family.

Returns

string —

$root.

Examples

foreach ($DOIs as $doi) { $dif = str_splitByRoot($doi, $root, ' * '); print "\n-- $doi \t= $dif"; }
** File not found : $root=str_getRoot($DOIs,1,5); **
