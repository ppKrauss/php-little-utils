file_csv()

file_csv(  $file,   $opt = array(),   $length,   $context = NULL) : array

Reads entire CSV file into an array (or associative array or pair of header-content arrays).

Like build-in file() function, but to CSV handling.

Parameters

$file

string filename.
$opt

not-null (can be empty) associative array of options (sep,enclosure,escape,head,assoc,limit)
$length

integer same as in fgetcsv().
$context

resource same as in fopen().

Returns

array —

(as head and assoc options).

assoc_join()

assoc_join(  $a,   $sep = '; ',   $pairSep = '=') : \mix

Joins a set of key-value pairs by $pairSep, and all pairs of the set by $sep.

NOTE: assoc_join($a,'&') is similar to http_build_query($a).

Parameters

$a

mix NULL or array (handdled as reference) to be joined.
$sep

string (default '; ') separator in the final string.
$pairSep

string (default '=') pair separator (joins key-val pair).

Returns

\mix —

NULL if $a is NULL, string if $a is array.
