See also
https://github.com/ppKrauss/php-little-utils MIT License Copyright (c) 2016 Peter Krauss isFile() at basiChecks.php Converts XML file (or XML string or DOMDocument) to JSON or to PHP array. Use direct SimpleXML-JSON convertion or jsonML (see https://github.com/mckamey/jsonml).
Parameters
$inputstring of filename or string of full XML or DOMDocument instance.
$out_phpboolean true if whant PHP output, else will be JSON.
$out_jsonmlboolean true if whant jsonML as outputformat, else will be SimpleXML-JSON.
$out_arrayboolean true if whant pure PHP array as output, else will be array with objects.
$flenLimitinteger 0 or size limit for filename, only for safe.
Returns
JSON string or array of XML mapped data.

NOTE: for other xml-json map strategies, see https://github.com/bramstein/xsltjson FUTURE function dom2json() can be used with XPath to select specific node or array of nodes.
